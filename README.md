## Project Overview
# Global Population Dashboard in Power BI

![Visualisation preview](./Visualisation%20preview.png)

## Files Included

- Power BI source file (.pbix)
- Dashboard PDF export
- Project documentation
  
The dashboard provides interactive analysis of population trends, urbanisation patterns, growth rates and demographic indicators across countries and regions between 1960 and 2050.

This project presents an interactive Power BI dashboard designed to analyse global population trends and demographic indicators across multiple countries and regions.

The dashboard combines historical population data and future projections to support data-driven insights into demographic change, urbanisation patterns and population growth.

## Key Features

- Population trends from 1960 to 2050
- Urban and rural population analysis
- Population growth rate calculations
- Country and regional comparisons
- Interactive slicers and filters
- Data visualisation using Power BI

## Tools and Technologies

- Power BI
- DAX
- Data Modelling
- Data Visualisation
- Demographic Analytics

## Business Value

The dashboard supports exploration of demographic changes that can assist governments, researchers and organisations in understanding long-term population patterns and planning future strategies.
